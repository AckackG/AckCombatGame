# UI 与交互控制 (UI & Controls)

包含玩家的输入处理、RTS 框选机制以及前端界面展示。

## 界面组成
- **主菜单**: 采用赛博朋克/战术终端风格 (`style.css`)，利用 CSS 动画实现故障文字特效和背景扫描线。包含战役模式、沙盘模式和数据库入口。
- **武器数据库 (`Database`)**: 遍历读取 `src/data/weapons_data.js`，动态渲染所有枪械的详细属性面板。
- **HUD 信息**: 悬浮在 Canvas 上方，显示 FPS、当前波次、下一波倒计时、场上单位状态，以及用于 Debug 的性能计数器。
- **战斗统计面板**: 暂停游戏并展示武器命中率、伤害效率、平均 DPS 与实战 DPS。

## 视角控制 (Viewport)
- 支持通过鼠标滚轮进行无极缩放 (Zoom)。
- 支持按住鼠标右键拖拽平移视角 (Pan)。
- 提供 `Reset View` 按钮一键复原缩放和平移状态。

## RTS 框选与控制 (`src/core/rts_control.js`)
- 玩家可按住鼠标左键在 Canvas 上拖拽形成选框，框选己方单位 (`color === player_color`)。
- 选中后，目标单位脚下会绘制选中指示器。
- 点击右键向选中单位下发移动指令，覆盖其原生 AI。接收到手动指令的单位获得一段“手动控制时间”（默认30秒），期间不会再受自动 AI 驱使乱跑。
- 右键点击敌方单位可下发强制攻击指令。
